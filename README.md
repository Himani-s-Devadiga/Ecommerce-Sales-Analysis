# Ecommerce-Sales-Analysis
E-commerce sales analysis project using Python and Power BI to study revenue, customer behavior, and regional performance.
# 📊 E-Commerce Sales Analysis Project

## Objective
Analyze 12 months of e-commerce sales data to identify trends, performance, and business insights.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Excel

## Key Insights
- Top selling category identified
- Monthly sales trend analyzed
- Regional performance compared
- Customer review distribution studied

## Dashboard
Includes KPI cards:
- Total Revenue
- Total Orders
- Average Order Value
- Top Category
- Best Month

## Files
- analysis.ipynb → Data cleaning & analysis
- dashboard.pbix → Power BI dashboard
- business_insights.txt → Final recommendations
